# chitchat
这是一个名为ChitChat的简易网上论坛，ChitChat实现网上论坛的关键特性：  
1. 在这个论坛里面，用户可以注册账号，并在登录之后发表新帖子又或者回复已有的帖子；
2. 未注册用户可以查看帖子，但是无法发表帖子或回复帖子。

## 应用设计
**客户端** → 发送HTTP请求 → **服务器**  
　　　　　　　　　　　　　　↓（处理HTTP请求）  
**客户端** ← 返回HTTP响应 ← **服务器**
